wowApet 派特網 網站

  • 2015-01-18

[前言...]
結束了將近半年的資策會訓練課程後。成功地展示了 avata 魚菜共生網站後(包含flash簡報，flash 手機ＡＰＰ)，經由ＣＳＳ高手 AMOS 老師介紹(css coke 作者)，到玉液科技公司上班。當初面試的時候，公司就有表明說是要使用Ruby on Rails開發網站。但是我完全是個網頁新手，連專題網站也只是用pure PHP。很感謝公司願意用我這個基本上完全沒有任何經驗的新鮮人。願意讓我在公司中邊做邊學ROR。派特網也是我的第一個ＲＯＲ做品。

派特網主要目的是做寵物平台，使用者可以搜尋任何跟寵物有關係的資料。並且會使用ＬＢＳ概念。讓使用者可以快速找到適合自己的資料。除了本身的資料庫以外，也搭配google search api以及google map api來增進資料豐富性。以下來簡略介紹派特網吧。


首頁搜尋。可以輸入關鍵字搜尋或是直接點擊搜尋，將會列出最靠近你的店家。


搜尋結果頁左邊側邊的按鈕可以點擊選擇特定類別。


搜尋結果頁上方按鈕可以根據條件過濾。


搜尋結果頁右邊地球圖示，點擊可進入地圖模式頁面。


另外有清單頁，小編報報頁，內容頁，建立頁面。


發現頁面有錯誤時，可以點擊回報問題。


會員系統包含個人資料，設定，通知，訊息，建立的店家。


也有架設一個論壇提供會員們討論的平台。


另外會不定期舉辦活動，贈與網站虛擬貨幣，然後提供物品供使用者兌換。


網站也有製作 api 供他站使用，包含 404寵物認養頁面 以及 部落格寵物認養專欄。

以上就是這個網站的大概內容。
這網站後台部分也很仔細，但不是我負責的，所以就沒有介紹了。
我主要負責的內容包含：
1.部分網頁切板。(主要在論壇頁，會員個人頁，寵物福利社，api設計)
2.將前端美術設計切板 HTML 轉為 ROR。
3.google api ＆＆ facebook api 串接
4.404認養以及部落格寵物認養功能。
5.其他大大小小功能。(回報問題，未讀通知，手動定位...等)


目前已經離職，不負責此專案了。實際狀況可能與以上描述不同。
